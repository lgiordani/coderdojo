Parte IV

La sezione segreta

Questa è l'ultima sezione del lavoro di oggi, e vogliamo farti fare alcuni passi un po' più avanzati, così da farti venire voglia di tornare la prossima volta!

Come ti sei sicuramente accorto, le pagine HTML dell'avventura che abbiamo scritto e che tu hai completato non sono proprio bellissime, sembrano un po' vuote con tutto quello spazio bianco. Insomma, sarebbe bello poterle rendere un po' più interessanti esteticamente, no?

Entra in scena il secondo linguaggio di cui parlavamo, chiamato CSS, che sta per Cascading Style Sheets, "fogli di stile a cascata". Per ora lasciamo stare cosa significhi "a cascata" e concentriamoci sul significato di "fogli di stile". Il linguaggio CSS ci permette di descrivere lo "stile" degli elementi HTML, ovvero tutte le cose come il colore, la dimensione, lo sfondo, la posizione nella pagina, eccetera.

In questa sezione quindi ti insegnamo ad inserire un foglio di stile nella pagina HTML e a cambiare alcune proprietà. La prossima volta faremo un lavoro un po' più approfondito sui CSS, per questa volta vediamo solamente le cose basilari.

Prima di tutto assicurati di avere i file dell'avventura che abbiamo scritto nella Parte 3. Se hai seguito le indicazioni che ti abbiamo dato precedentemente, i link che hai messo nelle pagine sono contenuti in una lista non ordinata. Se non lo hai fatto fallo ora, è molto importante per il risultato finale; se non ti ricordi come fare o se sei insicuro chiedi ad un mentor di aiutarti.

Ora apri nell'editor il primo file della tua avventura (quello dal nome 01.html nella cartella del tipo di ambientazione che hai scelto), e inserisci il foglio di stile che abbiamo preparato per te.

Nella parte iniziale del file, subito prima del tag </head> inserisci il codice
<link href="../css/style.css" rel="stylesheet" />.

Come vedi è un nuovo tipo di tag (<link>) che serve proprio ad inserire un link invisibile ad un altro file, che in questo caso si chiama style.css, poi lo apriremo e lo modificheremo un po'. Come vedi, inoltre, c'è l'indicazione stylesheet che indica proprio che si tratta di un foglio di stile. Il tag <link> è un tag vuoto proprio come <img> ed infatti lo abbiamo scritto con la notazione abbraviata.

Vogliamo vedere l'effetto di questa aggiunta?

Apri nel browser la prima pagina dell'avventura e... meraviglia! È tutto molto più bello, no? Se è tutto uguale a prima abbiamo sbagliato qualcosa, ricontrolla o chiedi ad un mentor di darti una mano.

Come è possibile che una singola riga di codice faccia tutto questo? Beh, come abbiamo detto sopra il tag <link> inserisce un altro file, quindi ora andiamo a cercarlo.

Nell'editor premi Ctrl+O e scegli di aprire il file ##base_dir##/part3/css/style.css.

Bene, questo che vedi è il linguaggio CSS. Come vedi è abbastanza uniforme, e se noti, ci sono i nomi dei tag HTML all'inizio delle varie sezioni (cioè le parti delimitate dalle parentesi graffe). Oggi non possiamo vedere tutto, ma iniziamo a fare una modifica e a vedere cosa succede. Se torni sul file 01.html che hai aperto prima noterai che la frase "La mia avventura" è inclusa in un tag <h1>, giusto? Bene, allora se vogliamo cambiare lo stile di questa parte dobbiamo andare nel file CSS e cercare se ci sono delle indicazioni per il tag h1. In effetti alla riga 20 del file CSS c'è proprio scritto h1 { seguito da alcune righe di comandi che per ora non conosciamo ancora, e alla riga 27 c'è la parentesi graffa che chiude la sezione.

Alla riga 23 troviamo l'indicazione color: white;, che, come dice il nome stesso, definisce il colore del contenuto del tag <h1>, quindi, per quanto ci riguarda, la frase "La mia avventura". Prima di addentrarci nelle spiegazioni tecniche proviamo a fare una modifica veloce.

Nell'editor apri il file style.css e cambia alla riga 23 il codice color: white; in color: yellow;. Poi vai sul browser e aggiorna la pagina. Cosa è successo? Se tutto è andato bene la frase "La mia avventura" è diventata gialla!

Perché la modifica che abbiamo fatto ha cambiato il colore della scritta da bianco a giallo? Ha cambiato il colore perché la regola è color: e quindi indica il colore del testo. Il colore è poi indicato dal nome white (bianco, in inglese). Quando abbiamo sostituito il codice con l'indicazione yellow abbiamo indicato che il colore della scritta è per l'appunto giallo (yellow in inglese).

Anche questa parte, per oggi è finita. Rimane ancora tantissimo da imparare, sia per il linguaggio HTML sia per il linguaggio CSS, ma ci sarà tempo nei prossimi incontri. Per ora facciamo un ripasso veloce di quello che abbiamo visto in questa ultima parte. Abbiamo imparato

  • cosa sia il linguaggio CSS e a cosa serva
  • come inserire un file CSS all'interno di un file HTML in modo che si applichi lo stile
  • che il file CSS è diviso in sezioni e che queste contengono delle regole
  • come cambiare il colore del testo
  • come si dice bianco e giallo in inglese (il giallo è importante, soprattutto per i semafori)

Questa volta siamo veramente alla fine!
